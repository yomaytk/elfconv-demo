BusyBox Web Demo

This is a browser-based BusyBox demo, compiled to WebAssembly using the elfconv binary translator.

The following commands have been verified to work in this demo:

  • pwd
  • mkdir
  • arch
  • uname
  • ls
  • ascii
  • basename
  • echo
  • cd
  • hostname
  • date
  • rm
  • rmdir
  • clear
  • tree
